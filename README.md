# NumPy Fundamentals 🧮

This repository contains my notes, practice code, and exercises while learning **NumPy**, one of the most important Python libraries for Data Science, Machine Learning, and Artificial Intelligence.

Rather than just finishing the chapter, I wanted to understand *why* NumPy is so widely used and how it makes working with data much more efficient than regular Python lists.

- Introduction to NumPy
- Why NumPy is faster than Python Lists
- Creating Arrays
- Array Properties
  - shape
  - size
  - ndim
  - dtype
  - itemsize
- Array Reshaping
- Indexing & Slicing
- Fancy Indexing
- Boolean Indexing
- Copy vs View
- Working with 1D, 2D & 3D Arrays
- Operations Along Axes
- Data Types & Type Casting
- Vectorization
- Broadcasting
- Mathematical Functions
- Statistical Functions
- Performance & Memory Comparison

## 🛠️ What I Learned

During this module, I learned how to:

✔ Work efficiently with multidimensional arrays

✔ Replace slow Python loops using vectorized operations

✔ Manipulate and reshape data for machine learning tasks

✔ Perform mathematical and statistical computations with just a few lines of code

✔ Understand why NumPy is the foundation of libraries like Pandas, TensorFlow, and PyTorch

## 🎯 Why This Matters

Before learning NumPy, arrays were just another data structure to me.

Now I understand that almost every Machine Learning model works with numerical data, and NumPy provides the fast and efficient operations needed to process that data.

This is one of those topics that makes you appreciate what's happening behind the scenes.
